About Me

Throughout my academic and professional journey, I have actively engaged in diverse technology projects that have strengthened my expertise in software development and data analysis. I developed a task management and dashboard web application using React and Tailwind CSS, featuring full CRUD functionality and interactive data visualizations powered by Chart.js.

I also built a task reminder system using PHP and MySQL, integrated with email notification features via PHPMailer, aimed at helping users stay organized and productive. In the field of data science, I applied data wrangling techniques and performed feature selection on real-world datasets using methods such as Random Forest and Univariate Selection.

My experience includes conducting statistical analysis using Chi-Square tests and Linear Regression, which I presented through comprehensive reports and insightful visualizations. Additionally, I created educational videos explaining key programming concepts such as the Dependency Inversion Principle (DIP), A* search algorithm, and tree data structures, supported by engaging visual tools.
